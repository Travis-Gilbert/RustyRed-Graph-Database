# Architecture Decision Records

Numbered, append-only records of architecturally significant decisions
in this repository. Each ADR explains the context, the decision, the
alternatives rejected, and the consequences.

Numbering starts at `0001`. ADRs are not renumbered; superseded ADRs
are marked as such and link to the replacement.
